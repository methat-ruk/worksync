# WorkSync Stack

Map reusable abstractions to this project as follows:

| Abstraction | WorkSync Selection |
|---|---|
| Web Framework | Next.js App Router |
| Frontend Language | TypeScript |
| Styling System | Tailwind CSS |
| UI Component Library | shadcn/ui |
| Backend Framework | NestJS |
| ORM / Data Access Layer | Prisma |
| Relational Database | PostgreSQL |
| Cache Layer | Redis |
| Queue System | BullMQ |
| Realtime Transport | Socket.IO |
| Object Storage | AWS S3 |
| API Documentation | Swagger |
| Container Runtime | Docker |
| Local Container Orchestration | Docker Compose |

Use framework-native capabilities where they satisfy requirements and match established repository patterns.
